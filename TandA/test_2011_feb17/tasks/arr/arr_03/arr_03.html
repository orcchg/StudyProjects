Сдвиг на 1

Ограничение по времени работы программы: 1 секунда.
Ограничение по памяти: 32 мегабайта.
Ввод из стандартного потока ввода (с клавиатуры).
Вывод в стандартный поток вывода (на экран).

Напечатайте входную последовательность, сдвинув ее циклически на 1 вправо.

Входные данные

Целое число 0 < N < 100. Затем N чисел, по модулю не превышающих 30000, через пробел.

Выходные данные

Циклически сдвинутая последовательность.

Примеры

Вход

Выход

5
2 17 18 -4 35

35 2 17 18 -4
