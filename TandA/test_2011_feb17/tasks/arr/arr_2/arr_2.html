Уникальные элементы

Ограничение по времени работы программы: 1 секунда.

Ограничение по памяти: 32 мегабайта.

Ввод из стандартного потока ввода (с клавиатуры).

Вывод в стандартный поток вывода (на экран).

Напечатайте только уникальные элементы данной последовательности чисел.

Входные данные

Целое число 0 > N < 232. Затем N чисел, по модулю не превышающих 30000, через пробел.

Выходные данные

Числа в том же порядке, кроме тех, что входят в последовательность более одного раза. Если уникальных чисел в последовательности нет, выведите -1

Примеры

Вход

Выход

4
1 7 5 1

7 5

5
1 5 5 1 1

-1
