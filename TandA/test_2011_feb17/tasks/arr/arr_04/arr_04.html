Сдвиг на k

Ограничение по времени работы программы: 1 секунда.
Ограничение по памяти: 32 мегабайта.
Ввод из стандартного потока ввода (с клавиатуры).
Вывод в стандартный поток вывода (на экран).

Напечатайте входную последовательность, сдвинув ее циклически на K вправо.

Входные данные

Целое число 0 < N < 100. Затем N чисел, по модулю не превышающих 30000, через пробел. Целое число 0 < K < 100.

Выходные данные

Циклически сдвинутая на K последовательность.

Выход
