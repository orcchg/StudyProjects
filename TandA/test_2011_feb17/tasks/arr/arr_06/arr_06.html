Нехороший множитель

Ограничение по времени работы программы: 1 секунда.
Ограничение по памяти: 32 мегабайта.
Ввод из стандартного потока ввода (с клавиатуры).
Вывод в стандартный поток вывода (на экран).

Напечатайте числа от 2 до N включительно, не кратные K. Оператором % пользоваться запрещается. Воспользуйтесь кодом этой задачи, когда будете реализовывать алгоритм "Решето Эратосфена".

Входные данные

Натуральные числа 2 < N < 1000 и 2 ≤ K < N

Выходные данные

Числа через пробел. Число К печатать.

Вход

Выход
