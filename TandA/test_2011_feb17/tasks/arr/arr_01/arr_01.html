Четные первыми

Ограничение по времени работы программы: 1 секунда.
Ограничение по памяти: 32 мегабайта.
Ввод из стандартного потока ввода (с клавиатуры).
Вывод в стандартный поток вывода (на экран).

Напечатайте из заданной последовательность чисел сначала четные числа, а затем нечетные.

Входные данные

Целое число 0 < N < 1000. Затем N чисел, по модулю не превышающих 30000, через пробел.

Выходные данные

Первая строка: четные числа из входной последовательности через пробел. Вторая строка: нечетные числа из входной последовательности через пробел.

Примеры

Вход

Выход

5
2 17 18 -4 5

2 18 -4
17 5
