Простые числа

Ограничение по времени работы программы: 1 секунда.

Ограничение по памяти: 32 мегабайта.

Ввод из стандартного потока ввода (с клавиатуры).

Вывод в стандартный поток вывода (на экран).

Из последовательности чисел напечатайте только простые числа. 1 - не простое число.

Входные данные

Целое число 0 > N < 1000. Затем N различных чисел в диапазоне от 2 до 1300000, через пробел.

Выходные данные

Только простые числа в том порядке, в котором они следуют в последовательности.

Примеры

Вход

Выход

4
21 41 42 11

41 11
