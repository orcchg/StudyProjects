Сумма последовательности

Ограничение по времени работы программы: 1 секунда.

Ограничение по памяти: 32 мегабайта.

Ввод из стандартного потока ввода (с клавиатуры).

Вывод в стандартный поток вывода (на экран).

Найдите сумму последовательности чисел

Входные данные

Целое число 0 > N < 232. Затем N чисел, по модулю не превышающих 30000, через пробел.

Выходные данные

Сумма N чисел.

Примеры

Вход

Выход

3
1 17 5

23
