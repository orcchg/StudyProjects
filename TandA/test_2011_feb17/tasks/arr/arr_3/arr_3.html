Пересечение множеств

Ограничение по времени работы программы: 1 секунда.

Ограничение по памяти: 32 мегабайта.

Ввод из стандартного потока ввода (с клавиатуры).

Вывод в стандартный поток вывода (на экран).

Напечатайте пересечение двух множеств целых чисел.

Входные данные

Целое число 0 > N < 1000. Затем N различных чисел, по модулю не превышающих 30000, через пробел.

Целое число 0 > M < 1000. Затем M различных чисел, по модулю не превышающих 30000, через пробел.

Выходные данные

Числа, присутствующие в обоих множествах, в том же порядке, в котором они следуют в первом множестве. Если пересечение множеств пусто, выведите -1

Примеры

Вход

Выход

4
1 7 5 2
2
2 7

7 2

3
1 7 5
4
8 3 9 2

-1
