Индекс элемента

Ограничение по времени работы программы: 1 секунда.
Ограничение по памяти: 32 мегабайта.
Ввод из стандартного потока ввода (с клавиатуры).
Вывод в стандартный поток вывода (на экран).

Найдите индекс числа в неупорядоченной числовой последовательности.

Входные данные

Целое число 0 < N < 1000. Затем N чисел, по модулю не превышающих 30000, через пробел. Далее число, так же по модулю не превышающее 30000.

Выходные данные

Индекс заданного числа в приведенной числовой последовательности или -1, если этого числа в последовательности нет. Если таких чисел несколько, выведите индекс первого совпадения (слева направо).

Примеры

Вход

Выход

7
2 17 0 18 -4 5 18
18

3

5
2 17 18 -4 5
38

-1
