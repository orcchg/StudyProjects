Границы факториала

Ограничение по времени работы программы: 1 секунда.

Ограничение по памяти: 32 мегабайта.

Ввод из стандартного потока ввода (с клавиатуры).

Вывод в стандартный поток вывода (на экран).

Дано число n. Вывести факториалы чисел до n! включительно. Указать, какой максимальный факториал представим в каком целочисленном беззнаковом типе

Входные данные

Дано n (натуральное число, не превышающее 30). Если факториал невычислим на данной платформе, то его выводить не нужно

Выходные данные

n!=факториал тип
