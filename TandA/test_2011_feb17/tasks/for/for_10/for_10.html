Число в обратном порядке

Ограничение по времени работы программы: 1 секунда.

Ограничение по памяти: 32 мегабайта.

Ввод из стандартного потока ввода (с клавиатуры).

Вывод в стандартный поток вывода (на экран).

Выведите число, состоящее из цифр данного числа x в обратном порядке. Ведущие нули выводить не нужно.

Входные данные

Целое положительное N < 232-1

Выходные данные

Число в обратном порядке без ведущих нулей.

Примеры

Вход

Выход

123

321

2100

12
