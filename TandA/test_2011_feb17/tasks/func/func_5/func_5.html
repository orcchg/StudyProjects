простой ковер

Напишите функции
void fill_rectangle(char a[20][20], int x, int y, int w, int h, char c);
void draw_rectangle(char a[20][20], int x, int y, int w, int h, char c);
Функция fill_rectangle рисует прямоугольник, draw_rectangle рисует прямоугольную рамку толщиной в один символ с в массиве а со сдвигом x колонок и y строк.
Рекомендуем написать функцию print(char a[20][20]), которая распечатывает массив.
