add bin

Напишите функцию void add(char a[], char b[], char res[]), которая складывает два длинных двоичных числа, записанных в виде строк и кладет результат в массив res. Исходные массивы a и b должны остаться без изменений.

Подсказка: воспользуйтесь ранее написанной функцией void revers(char a[], int n), расставляющую первые n элементов в массиве a в обратном порядке и напишите функцию int length(char a[]), которая возвращает количество цифр в бинарном числе. Функцию main реализовывать НЕ нужно. Слагаемые натуральные, не превосходят 2100.

Пример: после вызова функции add("10101010101", "11", res) в массиве res должна быть строка "10101011000"

Не забывайте, что строка должна оканчиваться символом '\0'
