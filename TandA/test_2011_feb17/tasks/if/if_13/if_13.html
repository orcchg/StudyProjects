Римская система счисления

Ограничение по времени работы программы: 1 секунда.

Ограничение по памяти: 32 мегабайта.

Ввод из стандартного потока ввода (с клавиатуры).

Вывод в стандартный поток вывода (на экран).

Дано число X. Требуется перевести это число в римскую систему счисления.

Входные данные

Дано число X в десятичной системе счисления (1 ≤ X ≤ 100).

Выходные данные

Выведите X в римской системе счисления.
