Принадлежит объединению множеств

Ограничение по времени работы программы: 1 секунда.

Ограничение по памяти: 32 мегабайта.

Ввод из стандартного потока ввода (с клавиатуры).

Вывод в стандартный поток вывода (на экран).

Найдите, принадлежит ли введенное целое число объединению множеств (-∞; 37) U [146; ∞)

Входные данные

Целое число по модулю меньше 30000

Выходные данные

YES или NO

Выход
