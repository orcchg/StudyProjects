Существует треугольник

Ограничение по времени работы программы: 1 секунда.

Ограничение по памяти: 32 мегабайта.

Ввод из стандартного потока ввода (с клавиатуры).

Вывод в стандартный поток вывода (на экран).

Даны три натуральных числа a, b, c. Определите, существует ли треугольник с такими сторонами.

Входные данные

Три натуральных числа, по модулю не больших 30000

Выходные данные

YES или NO

Примеры

Вход

Выход

3 4 5

YES
